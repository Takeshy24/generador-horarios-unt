# Frontend Skill

Objetivo:
Desarrollar interfaces modernas y mantener consistencia visual.

Reglas:

- Nunca tocar backend
- Nunca modificar APIs
- Nunca alterar contratos de datos
- Mantener TypeScript estricto
- Reutilizar componentes existentes
- Evitar duplicación
- Usar Tailwind
- Mantener diseño responsive

UI:

- Diseño estilo Linear/Vercel/Notion
- Espaciado consistente
- Componentes reutilizables
- Soporte modo oscuro
- Skeleton loading
- Estados vacíos
- Tablas con filtros
- Sidebar colapsable

Horarios:

- Vista calendario semanal
- Colores por curso
- Responsive

Código:

- Componentes pequeños
- Props tipadas
- Hooks separados
- Evitar lógica dentro JSX
- Mantener nombres descriptivos
